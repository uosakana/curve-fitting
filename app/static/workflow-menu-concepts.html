三个方案都模拟首页 mega menu 的展开层。鼠标放到每一步上，看解释文字是否稳定、是否轻快，以及整体有没有压住首页的视觉。

Option A 轻量流程条
最像 mega menu

Raw data becomes a reviewed fit, then a reusable record.

Option B 压缩卡片
保留卡片感

Five compact cards keep the current structure, but remove the tall empty lower half.

Option C 圆点流程图
更像流程图

A thin path makes the workflow more visual, but it is also the most decorative option.
